# LEO AI

A real-time AI avatar built with Three.js, React, and OpenAI.

## 🎯 Vision

This project explores next-gen conversational interfaces by combining:

- AI reasoning
- real-time 3D rendering
- voice interaction

Inspired by future human-computer interaction systems.

## ✨ Features

- 🎙️ Voice interaction (speech-to-text + text-to-speech)
- 🧠 AI-powered conversation
- 🎭 Facial animation using morph targets
- 🌌 Futuristic 3D environment
- 🎧 Audio-reactive visuals
- 🧬 AI “brain” visualization (WIP)

## 🧱 Tech Stack

Frontend:

- React + TypeScript + Vite
- Three.js (React Three Fiber)

Backend:

- Node.js + Express
- OpenAI API

## 🚀 Getting Started

### 1. Clone

````bash
git clone https://github.com/YOUR_USERNAME/neuroface-ai.git
cd neuroface-ai

### 2. Install frontend dependencies

```bash
cd client
npm install
npm run dev
````

### 3. Install backend dependencies

```bash
cd server
npm install
```

### 4. Create .env

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

### 5. Run frontend

```bash
npm run dev
```

### 6. Run Backend

```bash
npm run dev
```
